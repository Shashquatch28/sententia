# Render Deployment Checklist

This deployment runs HireIQ as one Render web service:

- Static frontend from `frontend/`
- API backend from `server.py`
- Precomputed data from `precomputed/`
- SQLite demo knowledge store seeded at startup

## Required Render Environment Variables

Render reads `render.yaml`, which defaults the service to Gemini. Set the secret
API key before the final demo:

```text
GEMINI_API_KEY=<your Gemini API key>
```

`HIQ_SECRET_KEY` is generated by the Render blueprint. You can override it with
any long random string if needed.

Alternative providers:

```text
LLM_PROVIDER=groq
LLM_MODEL=llama-3.1-8b-instant
GROQ_API_KEY=<your Groq API key>
```

```text
LLM_PROVIDER=openai_compatible
LLM_MODEL=<provider model>
LLM_API_BASE=<provider base URL ending in /v1>
LLM_API_KEY=<provider API key>
```

## Deploy Steps

1. Push the repo to GitHub.
2. In Render, create a new Blueprint from the repo.
3. Confirm Render detects `render.yaml`.
4. Fill `GEMINI_API_KEY`.
5. Deploy the `sententia-hireiq` web service.
6. Open the service URL.

## Demo Verification

Run these checks from the public Render URL:

- [ ] `/` opens the sign-in screen.
- [ ] `Continue as demo` signs in.
- [ ] `/index.html` loads the dashboard.
- [ ] Candidate shortlist loads from `precomputed/demo_data.json`.
- [ ] Candidate detail rail opens.
- [ ] Pipeline advance/set-aside buttons persist during the live session.
- [ ] `/api/health` returns `"status": "ok"` and nonzero knowledge-store counts.
- [ ] Copilot answers a recruiter question.
- [ ] Candidate rail `AI what-if` returns an LLM-backed scenario response.
- [ ] Candidate rail `Draft outreach` returns a generated email.
- [ ] Incognito browser can repeat the flow without localhost errors.

## Known Hackathon Tradeoffs

- SQLite is used as a local demo store. It is fine for the judging session, but decisions can reset when the Render service restarts or redeploys.
- The LLM features require a hosted provider key. Without it, the UI still loads, but Copilot, simulation, and outreach report that no LLM provider is configured.
- Render free services can spin down after inactivity. Open the app a few minutes before judging so the first request has time to wake the service.
